# Gravity Balls

Live demo: https://pkspeleo.github.io/gravitation/

A single-file HTML toy that simulates simple gravitational interaction between glowing balls.  
Color represents **speed relative to the current average**, and soft trails show their motion history.

## How to use

- Press `=` / `+` (or numpad `+`) to add a new ball
- Press `-` (or numpad `-`) to remove a ball
- On mobile:
    - Tap with **one finger** to add a ball
    - Tap with **two fingers** to remove a ball
